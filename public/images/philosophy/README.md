# Philosophy Carousel Images

Place 4 images in this folder for the philosophy section carousel.

**Image Requirements:**
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Recommended size: 1200x800px or similar aspect ratio
- Images will be automatically cropped to fit the carousel container

**Naming Convention:**
Name your images:
- `philosophy-1.jpg` (or .png)
- `philosophy-2.jpg` (or .png)
- `philosophy-3.jpg` (or .png)
- `philosophy-4.jpg` (or .png)

The carousel will automatically rotate through all 4 images in order.
