## Quick Tips

1. **Wheeler Crossings:** All images are required for the page to display properly
2. **Claro Comps:** Images are optional - page works fine with current placeholders
3. **File formats:** Use JPG for photos, PNG for charts/tables with text
4. **Optimization:** Compress images for web (aim for <500KB each)
5. **Resolution:** Hero/closing images should be 1920px+ width
